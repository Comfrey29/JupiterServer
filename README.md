## Overview

Jupiter Server is a lightweight **Linux-based operating system** designed specifically for servers, web projects, and containerized applications.  
This version is **terminal-only** and includes essential tools for deploying services securely and efficiently.

---

## Key Features

- Minimal, terminal-only system  
- Pre-installed packages:
  - Python 3, Node.js, Go, Git, Docker  
  - Basic Linux commands: `curl`, `wget`, `ping`  
- Lightweight web server setup (Nginx)  
- Secure SSH enabled with root login disabled  
- Fail2Ban for brute-force protection  
- Custom login banner: "Jupiter Server"  
- Basic system monitoring with `htop`

---

## Minimum Requirements

- Architecture: 64-bit (amd64)  
- RAM: ≥2 GB recommended  
- Disk: ≥10 GB  
- Active network connection for updates and Docker
